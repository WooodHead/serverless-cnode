# serverless-cnode

A cnodejs client using Next.js deploy to Serverless

## Flow

![Deploy Flow](./docs/ssr-deploy-flow.png)

## Feature

- [x] Typescript
- [x] Next.js
- [x] Express Server
- [x] LRU Cache
- [x] Static CDN Deployment
- [x] Serverless Framework, using Next.js component

## Setup

```bash
$ npm install

$ npm run dev
```

## Build

```bash
$ npm run build
```

## Deploy

```bash
$ npm run deploy
```
